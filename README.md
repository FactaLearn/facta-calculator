# Facta Calculator

## Como usar?

Clone o repositório para sua máquina

```
git clone https://github.com/FactaLearn/facta-calculator.git
cd facta-calculator
```

Baixe as deps e rode o server

```
npm install
npm start
```

## Testes

```
npm test
```
